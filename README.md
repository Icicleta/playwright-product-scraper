# Playwright Product Scraper

Python web scraping project using Playwright to extract product data from dynamic websites.

## Features

- Scrapes product information from an e-commerce site
- Handles dynamic content using Playwright
- Extracts:
  - Title
  - Price
  - Stock availability
- Parses HTML content
- Exports data to CSV

## Tech Stack

- Python 3
- Playwright
- BeautifulSoup

## Installation

```bash
pip install -r requirements.txt
playwright install
